Standard Installation / 标准安装

Service Info / 服务信息

Install home fixtures (< 5kg) on walls/ceilings/floors. Any Quantity/Types of items. Service extension will be charged at 30-min blocks.

Standard Installation does not involve water, electrical, human or structrual safety. Else, categorized as Technical Installation.

Minimum 1 Hour booking.

在墙面/天花板/地。普通安装，物件重量不能超出5公斤. 安装物种，数量不限。如需加时，便以 30 分钟 "块" 计算。

安装过程不涉及水，电, 人身或屋子结构安全。诺牵涉到，就所属技术安装。

最短 1 小时预约。

Examples / 例子:
  • Wall Paintings / 壁画.
  • Hooks / 挂钩.
  • Battery Clocks / 电池钟.
  • Clothes Hangers / 衣架.
  • Key Holders / 钥匙扣架.
  • Bathroom Shelves / 浴室架.
  • Toilet Roll Holders / 厕纸吊勾.
  • Dismantle or install curtain / 拆或安装窗帘布 (< 3m) etc.

Technical Installation / 技术安装

Service Info / 服务信息

Installation involving / 安装涉及到:

  • Electricity / 电.
  • Water or fluid / 水.
  • Direct human safety / 人身安全.
  • Structural safety / 结构安全.
  • Items at heights / 高处物件 (> 3m).
  • Heavy items / 重物件 (> 8kg).
  • Glass & Mirror / 玻璃类.
  • Assembly tasks / 组装物件.
  • Signal, Radio Frequency, Internet, LAN cabling / 网路，讯号电缆等.
Examples / 例子:
  • Wall or Ceiling Fans / 风扇.
  • Water Heaters / 热水器.
  • Lights / 灯饰.
  • Door Bells / 门铃.
  • High curtain systems / 高窗帘.
  • Bulb Change / 换灯泡.
  • Replace faulty electrical sockets / 更换电插头等等.

Material Picking Fee (Optional) / 代购费 (可选择)

All installation services exclude price of new part(s).

A Material Picking Fee of RM45 to be charged, if Client want Blooms to purchase the item (at market price, with no markup) on behalf.

安装服务不包括替代件（如灯管，热水器，插头等）。如要家盟 (Blooms) 代购，公司会另收 RM45 代购费。
